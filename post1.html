COMO USAR O PACOTE FILEPATHS PARA MOVER ARQUIVOS EM JULIA?

Mover arquivos é uma tarefa comum no dia a dia de muitos profissionais, especialmente aqueles que trabalham com dados.

Autor: Dante Bertuzzi. Tempo de Leitura: 5 minutos. Data de Publicação: 24 de fevereiro de 2024.
Tags: Julia, Coding, Programming

Mover e organizar arquivos são tarefas essenciais para profissionais que lidam diariamente com dados. Em ambientes como Julia, a escolha das ferramentas certas pode fazer toda a diferença na eficiência do processo.

Neste contexto, a biblioteca FilePaths surge como uma solução intuitiva e eficaz para a manipulação de caminhos de arquivos em Julia. Este artigo visa orientar sobre a utilização do pacote FilePaths, proporcionando uma visão abrangente desde a sua instalação até exemplos práticos de como mover arquivos entre diretórios.

Ao explorarmos as funcionalidades desta biblioteca, os profissionais que trabalham com dados encontrarão uma ferramenta valiosa para simplificar suas operações diárias, permitindo-lhes focar em aspectos mais cruciais de seus projetos.

Instalando o pacote: Para usar o pacote FilePaths, é necessário instalá-lo primeiro. O comando abaixo pode ser usado para instalar o pacote:

using Pkg Pkg.add("FilePaths")

using FilePaths: Esta linha está incluindo o módulo FilePaths, que fornece funcionalidades relacionadas a manipulação de caminhos de arquivos.

pasta = "C:/users/..."

Aqui, está sendo atribuído um caminho de diretório à variável pasta. Este é o diretório onde os arquivos serão analisados e movidos.

for arquivo in readdir(pasta)

Um loop for é iniciado para iterar sobre cada arquivo no diretório especificado por pasta. A função readdir é usada para obter a lista de arquivos no diretório.

if occursin(".xlsx", arquivo)

Este bloco if verifica se o nome do arquivo contém a substring ".xlsx". Isso é usado para garantir que apenas arquivos Excel sejam considerados. 


                    if occursin("Jan", arquivo)

                        mv(joinpath(pasta, arquivo), joinpath(pasta, "Jan", arquivo))

Dentro do primeiro bloco if, há um segundo bloco if que verifica se a substring "Jan" está presente no nome do arquivo. Se for verdadeiro, o arquivo será movido para uma pasta chamada "Jan" dentro do diretório original. 


                    if occursin("Fev", arquivo)

                        mv(joinpath(pasta, arquivo), joinpath(pasta, "Fev", arquivo))

Se a condição anterior não for atendida, este bloco elseif verifica se a substring "Fev" está presente no nome do arquivo. Se for verdadeiro, o arquivo será movido para uma pasta chamada "Fev" dentro do diretório original. 


                    if occursin("Mar", arquivo)

                        mv(joinpath(pasta, arquivo), joinpath(pasta, "Mar", arquivo))

Similar ao bloco anterior, este bloco elseif verifica se a substring "Mar" está presente no nome do arquivo. Se for verdadeiro, o arquivo será movido para uma pasta chamada "Mar" dentro do diretório original.

end

Este end marca o fim das condições dentro do loop for. 


                    
                        using FilePaths
                        
                        pasta = "C:/users/..."
        
                        for arquivo in readdir(pasta)
                            if occursin(".xlsx", arquivo)
                                if occursin("Jan", arquivo)
                                    # mover para a pasta de janeiro
                                    mv(joinpath(pasta, arquivo), joinpath(pasta, "Jan", arquivo))
                                elseif occursin("Fev", arquivo)
                                    # mover para a pasta de fevereiro
                                    mv(joinpath(pasta, arquivo), joinpath(pasta, "Fev", arquivo))
                                elseif occursin("Mar", arquivo)
                                    # mover para a pasta de março
                                    mv(joinpath(pasta, arquivo), joinpath(pasta, "Mar", arquivo))
                                end
                            end
                        end
